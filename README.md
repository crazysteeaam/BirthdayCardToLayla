# BirthdayCardToLayla

一个为寿星 **Layla** 制作的移动端生日惊喜 H5 页面。  
整体体验采用“专属剧场”叙事：先验证身份，再开场烟花、分幕祝福，最后揭晓“音乐剧票根风格蛋糕”。

## 功能亮点

- 深色舞台氛围 + 光效粒子 + Canvas 烟花
- 趣味验证页（选择“今天的主角”）
- 分幕式祝福卡片，逐步推进情绪节奏
- 票根式蛋糕主视觉（参考剧场票根版式）
- 可配置的文案、票面信息、候选项、背景音乐开关
- 移动端优先适配（扫码直开）

## 页面流程

1. 加载页（氛围预热）
2. 趣味验证页（选择寿星）
3. 开场页（烟花 + 开场文案）
4. 创意祝福内容页（分幕浏览）
5. 蛋糕入口页（票根按钮）
6. 票根蛋糕展示页（核心视觉）
7. 收尾页（最终祝福）

## 项目结构

```text
.
├── index.html     # 页面结构与场景模块
├── styles.css     # 整体视觉风格与动效样式
├── script.js      # 场景状态机、配置项、交互与烟花逻辑

└── 票根.jpg        # 票根参考图
```

## 可配置项

主要在 `script.js` 顶部 `CONFIG` 中修改：

- `birthdayName`：寿星英文名（当前为 `Layla`）
- `birthdayDate`：生日日期
- `storyLines`：分幕祝福文案
- `ticket`：票根信息（票号、演出季、演出名、地点、区域、座位、时间、票种、祝福语）
- `verifyOptions`：验证候选项（支持 emoji 或 `image` 图片 URL）
- `music.src`：背景音乐地址（为空则只显示开关提示）

## 本地使用

直接在浏览器打开 `index.html` 即可运行。  
建议使用本地静态服务访问，避免部分浏览器对资源与音频策略限制。

示例（任选其一）：

```bash
python3 -m http.server 8080
```

然后访问：`http://localhost:8080`

## 部署建议

- 部署为可公开访问的 HTTPS 链接
- 生成二维码用于现场扫码
- 若在微信环境打开，建议提前实机测试音频触发与动画流畅度
