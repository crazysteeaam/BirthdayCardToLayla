# BirthdayCardToLayla

这不是一个普通网页，  
而是一份写给 **Layla** 的生日礼物。

从“请选择今天的主角”开始，  
到烟花、返场、票根蛋糕，  
每一幕都在说同一件事：

**你值得被认真庆祝。**

---

愿你在新的一岁里，  
继续被热爱包围，被好运偏爱，  
也继续做那个会发光、会心动、会让身边人变得温柔的人。

愿每一次上场都有掌声，  
每一次谢幕都有人等你返场。

**Layla，生日快乐。**

---

如果你打开了这个仓库，  
谢谢你一起见证这份小小但真诚的心意。
